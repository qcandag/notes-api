# A notes api that provides CRUD & Powerful search options and 3rd party authentication.

Made with Node.Js & Express.Js

## Installation

Clone the project

```bash
https://github.com/qcandag/notes-api.git
```

Move into project

```bash
cd notes-api
```

Install packages

```bash
npm install
```

## Run

```bash
npm run start
```
